# Minimalistisch voorbeeld van links maken binnen een image

In dit voorbeeld laten we zien hoe je binnen een afbeelding (image) links kunt maken naar verschillende secties van de afbeelding zelf. Dit kan handig zijn voor interactieve afbeeldingen zoals kaarten, diagrammen of infographics.

Zie W3schools voorbeeld: [text](https://www.w3schools.com/html/html_images_imagemap.asp)
